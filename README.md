# 独立开发者读物

独立开发者相当于独自创业，这不是社会主流，我们很难被动地学习到有益于独立开发的知识和观点。本仓库收录一些独立开发相关的杂文，包括AI独立开发、独立游戏开发等，希望对于走上独立开发之路的人有所帮助。
